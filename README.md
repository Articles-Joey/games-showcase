# Games Showcase

![Preview](/public/img/preview.webp)

A 3D collection of games I developed, this serves as a portfolio/another 3D example project/game launcher.

This started as a portfolio and launcher for games published by Articles Media. Primary goal is portfolio/launcher. Secondary goals is being able to use the launcher locally for other game stores as well.

## Getting Started

First, run the development server:

```bash
npm run dev
```

## Game Store Support

One of the features on top of just a showcase of the Articles Media games is that this can be used to showcase your own pc game collection. Manually add games via json or add your game library locations to see them in the launcher.
